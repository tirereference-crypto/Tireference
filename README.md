# TireReference

TireReference is a free tire fitment and tire size reference platform.

## Features

- Tire Size Calculator
- Plus Size Calculator
- Tire Comparison
- Tire Width Calculator
- Speedometer Calculator
- Wheel Offset Calculator
- Bolt Pattern Guides
- Tire Guides
- Fitment Guides

Website:
https://tirereference.com

Built with

- Astro
- React
- TypeScript
- Tailwind CSS
